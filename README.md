# nhl-attendance-analytics

## Project Overview
This Power BI project analyzes NHL regular season attendance to understand:
- Which teams have the highest and lowest attendance
- Whether team performance impacts attendance
- What factors influence attendance trends
- Which fan bases are most loyal

## Tools Used
- Power BI
- Power Query
- DAX
- Python
- NHL API
- GitHub

## Data Model
The project uses a STAR schema with:
- FactGames
- DimTeams
- DimArena
- DimDate

## Business Questions
1. Which teams lead the league in attendance?
2. Does winning drive attendance?
3. How do day of week and month affect turnout?
4. Which markets overperform relative to their size?
